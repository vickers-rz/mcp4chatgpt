# ChatGPT 学生使用指南提示词总结

来源：ChatGPT 学生使用指南（https://chatgpt.com/use-cases/students/）

## 一、页面定位

该页面收集了面向大学生的 100 个 ChatGPT 使用场景，覆盖：

- 学习辅助
- 论文与研究
- 知识理解
- 复习备考
- 职业发展
- 生活规划
- 思维训练

这些提示词的共同特点：不是简单要求 ChatGPT 给答案，而是把 ChatGPT 作为：

- 学习伙伴
- 教练
- 教授
- 研究助手
- 模拟对象
- 思维挑战者

---

# 二、核心提示词模式分类

## 1. 教授/评分者模式

### 用途

让 ChatGPT 扮演专业评价者，提供标准化反馈。

### 示例

> 请以教授的身份，根据评分细则为我的论文打分。

### 可迁移模板

```
请扮演[专业角色]，根据[评价标准]评估我的[作品/方案/回答]。
请指出优点、问题和改进方向。
```

适用：

- 论文
- 代码审查
- 演讲
- 设计方案
- 商业计划

---

# 2. 学习伙伴模式

## 核心思想

不要直接讲答案，而是通过互动帮助理解。

示例：

> 我会逐一讲解我的幻灯片，希望你像学习伙伴一样帮助我理解。

### 模板

```
请作为我的学习伙伴。
我会尝试解释[内容]，请通过提问帮助我发现理解漏洞。
不要直接给答案，优先引导思考。
```

适合：

- 主动学习
- 费曼学习法
- 复习知识

---

# 3. 苏格拉底提问模式

核心：让 ChatGPT 提问，而不是教学。

示例：

> 通过提出开放性问题来引导我，鼓励反思、批判性思维和知识应用。

模板：

```
请采用苏格拉底教学法。
通过连续提问帮助我理解[主题]。
除非我请求，否则不要直接解释答案。
```

---

# 4. 根据状态调整教学模式

示例：

> 我今天感到极度缺乏动力、精疲力尽，请基于这种状态帮助我学习。

核心思想：

教学策略应该适应人的状态。

模板：

```
我的当前状态是：[疲惫/焦虑/没有动力/时间紧张]。
请调整教学方式，帮助我完成[目标]。
```

# 三、最佳实践总结

这些学生提示词体现出的高级 ChatGPT 使用方式：

## 1. 不只是问答案

低级用法：

> 告诉我答案。

高级用法：

> 帮助我理解、验证、练习和改进。

---

## 2. 明确角色

有效提示词通常包含：

```
请扮演……
```

例如：

- 教授
- 教练
- 导师
- 学习伙伴
- 审稿人
- 面试官

---

## 3. 明确过程

优秀提示词描述：

- 如何互动
- 输出格式
- 判断标准
- 限制条件

---

## 4. 让 AI 成为认知伙伴

最高价值场景不是替代思考，而是：

- 提问
- 反馈
- 挑战
- 结构化
- 模拟
- 迭代

---
